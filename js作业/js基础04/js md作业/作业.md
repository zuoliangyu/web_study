### 拓展题

**目标：** 封装一个 every 函数， 用于检测数组所有元素是否都满足条件。

**需求：**检测数组里面的所有元素是否都大于10，如果满足则返回true， 如果不满足则返回 false

例如：

```html
<script>
  function every(ele, arr = []) {
      // 内部处理   
      let re = true
      for(let i = 0;i < arr.length; i++){
         if(arr[i] < 10){
             re = false 
             break
         }
          return re
      }
  }
  console.log(every(10, [10, 20, 30]))   // true
  console.log(every(10, [10, 3, 30]))   // false
</script>
```
